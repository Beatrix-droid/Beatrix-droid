Welcome to the UK Wild Plant Foraging API!

This is an API that communicates with a database that holds information on wild UK plants

that can be foraged. Use this API to retrieve information about the wild UK plants.

When retrieving information about a plant, the API will provide the user with the following information on the plant:

  • The plant's Latin name.
  • One of the plant's common names (some plants have multiple names depending that vary depending on region and dialect, the api provides the plant most "well known" common mame)
  • The plant's edible parts
  • A URL that links to an image of the Plant's leaves.
  • A URL that links to an image of the Plant's flowers.
  • A URL that links to an image of the Plant's stem/bark.
  • A URL that links to an image of the Plant's fruits/seeds.

There are 3 types of endpoints that you can use on this API

  • get-all-plants: returns information on all of the plants that are in this database.
  • get-plant-by-scientific name: accepts a scientific name as a parameter and, if the latin name is present in the database, returns information on the plant that has that scientific name.
  • get-plant-by-common name: accepts a common name as a parameter and, if the latin name is present in the database, returns information on the plant that has that common name.

Please note that for the last endpoint, because only one common name per plant is registered in the database, you may not be able to find a plant with a particular common name, if that name does not match the one present in the database. In those cases please use the get-by-scientific-name to get information on the plant you are looking for.
